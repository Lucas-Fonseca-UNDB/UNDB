CSS

CSS (Cascading Style Sheets) define o estilo e a aparência de um site. Ele é usado para adicionar cores, fontes e layouts às páginas.

Exemplo básico de CSS:

<style>
body {
    font-family: Arial, sans-serif;
    background-color: #f0f8ff;
    color: #333;
}
h1 {
    color: #4CAF50;
    text-align: center;
}
p {
    line-height: 1.6;
}
</style>

Principais Propriedades do CSS:

  • color: Cor do texto
  • background-color: Cor de fundo
  • margin e padding: Espaçamento externo e interno
  • border: Bordas
  • font-family: Fonte do texto
